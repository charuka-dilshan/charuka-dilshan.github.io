Detailed Course Notes

1. The Internet vs. The World Wide Web

These two terms are often used interchangeably, but they are not the same.

  • The Internet: A vast, global **network of computers** that connects millions of people. [cite_start]It is the hardware and infrastructure[cite: 2209].
  • The World Wide Web (Web): A **multimedia-based collection of information, services, and Web sites** that is supported by the Internet. [cite_start]It is the content that flows over the network[cite: 2208].

2. Web Sites, Pages, and Addresses

    [cite_start]
  • Web Site: A specific location on the Web where you visit and gather information (e.g., Google's site)[cite: 2213].
    • [cite_start]
  • Web Site Address (Domain Name): A unique name that identifies a specific computer on the Web (e.g., `www.google.com`) [cite: 2214-2216].
    • [cite_start]
  • Web Page: A specific portion of a Web site that deals with a certain topic (e.g., the "About Us" page)[cite: 2222].
    • [cite_start]
  • URL (Uniform Resource Locator): The full address for a specific web page (e.g., `http://www.google.com/about`)[cite: 2223].

Understanding Addresses

A URL like `http://www.google.com` can be broken down:

  • http://: The protocol (Hypertext Transfer Protocol). [cite_start]This part is often not required to be typed anymore [cite: 2228-2229].
  • www: Stands for World Wide Web.
  • google.com: The domain name.
  • .com: The **Top-Level Domain (TLD)**. [cite_start]This extension identifies the type of organization [cite: 2234-2235].

3. Core Internet Technologies

Web Browser Software

The software that enables you to surf the Web. [cite_start]Popular examples include Google Chrome, Microsoft Edge, and Mozilla Firefox [cite: 2247, 2249-2252].

Internet Infrastructure

    [cite_start]
  • Internet Backbone: The major set of high-speed connections for computers on the Internet[cite: 2267].
  • Network Access Point (NAP): A point on the Internet where several connections converge. [cite_start]This is where routing computers are located [cite: 2268-2269].
    • [cite_start]
  • Network Service Provider (NSP): A company that owns and maintains the routing computers at NAPs and the high-speed lines connecting them (e.g., AT&T)[cite: 2271].
    • [cite_start]
  • Internet Service Provider (ISP): A company that provides individuals and businesses access to the Internet (e.g., Dialog, SLT)[cite: 2306].

Client vs. Server

    [cite_start]
  • Client Computer: The computer you use to access the Internet (e.g., your laptop or phone)[cite: 2312].
    • [cite_start]
  • Internet Server Computer: A computer that provides information and services on the Internet[cite: 2313].
[cite_start]

Types of Internet Servers [cite: 2315, 2319-2323]

Server Type Function Example
Web Server Provides information/services to surfers. Accessing `www.ebay.com`.
Mail Server Provides e-mail services and accounts. Using Gmail or Outlook.
FTP Server Maintains a collection of files to download. Downloading software or game files.
IRC Server Supports discussion groups and chat rooms. Participating in a live chat forum.

4. Communication Protocols

[cite_start]

A **protocol** is a set of rules that computers follow to transfer information[cite: 2327].

    [cite_start]
  • TCP/IP (Transmission Control Protocol/Internet Protocol): The primary protocol for transmitting all information over the Internet[cite: 2329].
    • [cite_start]
  • HTTP (Hypertext Transfer Protocol): The protocol that supports the movement of information (web pages) over the Web[cite: 2330].
    • [cite_start]
  • FTP (File Transfer Protocol): The protocol that allows you to transfer files from one computer to another[cite: 2331].

5. Search Engines

[cite_start]

A **search engine** is a software system designed to search for information on the World Wide Web (e.g., Google, Bing, Yahoo!) [cite: 2335, 2342-2348].

    [cite_start]
  • Search results are presented in a list called **SERPs** (Search Engine Results Pages)[cite: 2336].
  • Search engines are different from **web directories**, which are maintained by human editors. [cite_start]Search engines maintain real-time information by running an **algorithm** on a **web crawler**[cite: 2338].
